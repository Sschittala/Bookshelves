# Bookshelves

Digital catalog and circulation system managing members, books, loans, and holds. Tracks unique titles with multiple authors, monitors physical copy availability and condition, maintains FIFO hold queues for unavailable books.

# Technologies used

**Frontend:** React, Typescript
**Backend:** Python, Flask, SQLite
**Data Management:** SQLite
