# About

This document is documentation and instruction on how to launch the app.

# Getting started

1. Launch the environment from the backend folder by using `. /env/bin/activate`.
2. Make sure the requirements are installed via `pip install -r requirements.txt`.
3. Run the server via `flask --app app/main run --debug` and follow the address given. For more launch option check out the [official Flask quickstart](https://flask.palletsprojects.com/en/stable/quickstart/) guide.
